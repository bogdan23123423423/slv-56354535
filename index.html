Amnezia VPN — бесплатное приложение с открытым исходным кодом для создания VPN на собственном сервере (self-hosted). Сервер полностью в вашем распоряжении — доступ к нему есть только у вас, без посредников в лице обычного VPN-провайдера.

Приложение поддерживает сразу несколько протоколов:

  • OpenVPN и WireGuard — классические, проверенные протоколы;
  • AmneziaWG — собственная разработка Amnezia на основе WireGuard, которая маскирует VPN-трафик под обычный интернет-трафик;
  • XRay/VLESS Reality, ShadowSocks, IKEv2 — протоколы для обхода блокировок в странах с высоким уровнем цензуры.

Код клиента и сервера открыт для проверки, сервис не ведёт логов активности, а безопасность приложения регулярно проверяют независимые аудиторы. Платить нужно только за аренду собственного сервера (VPS) у любого хостинг-провайдера — само приложение бесплатно.
